# 공급업체 분석 보고서 - 설치 가이드

## 🚀 빠른 시작

### 1단계: CSV 파일 복사 (필수!)

앱을 실행하기 전에 CSV 파일을 public 폴더에 복사해야 합니다:

```bash
# Windows 명령 프롬프트
copy supplier_info-export-2025-11-10_16-16-36.csv public\
copy evaluations-export-2025-11-11_15-25-24.csv public\
```

또는 파일 탐색기에서:
1. 프로젝트 루트의 두 CSV 파일 선택
2. 복사 (Ctrl+C)
3. public 폴더로 이동
4. 붙여넣기 (Ctrl+V)

### 2단계: 앱 실행

#### 방법 A: 더블클릭 (추천)
1. `start.bat` 파일을 더블클릭
2. 자동으로 설치 및 실행됩니다

#### 방법 B: 수동 실행
```bash
npm install
npm run dev
```

### 3단계: 브라우저에서 확인
- 자동으로 `http://localhost:5173` 열림
- 데이터가 자동으로 로드됩니다!

## 📊 사용 방법

### 일반 사용자
- 앱 접속 시 자동으로 모든 공급업체 데이터 로드
- 공급업체 목록 조회 및 보고서 확인

### 관리자 (비밀번호: 0070)
1. "관리자 로그인" 클릭
2. 비밀번호 `0070` 입력
3. 새로운 CSV 파일 업로드 가능
4. 보고서 내용 수정 가능

## 🎨 주요 기능

- ✅ 자동 데이터 로드 (CSV 파일 업로드 불필요)
- ✅ Paperozi 폰트 적용
- ✅ 2개 CSV 파일 통합 분석
  - 업체 정보 (supplier_info)
  - 평가 결과 (evaluations)
- ✅ 관리자 인증 시스템
- ✅ 분석 내용 수정 기능
- ✅ 현업팀 평가 결과 표시

## ⚠️ 문제 해결

### CSV 파일을 찾을 수 없습니다
→ public 폴더에 CSV 파일이 있는지 확인하세요

### 데이터가 로드되지 않습니다
→ 브라우저 콘솔(F12)에서 오류 확인
→ CSV 파일명이 정확한지 확인

### 관리자 로그인이 안됩니다
→ 비밀번호: `0070` (숫자 0070)

## 🔄 데이터 업데이트

### 관리자가 새 데이터 업로드 시:
1. 관리자 로그인
2. "업체정보" 버튼 → 새 CSV 업로드
3. "평가결과" 버튼 → 새 CSV 업로드
4. 데이터 자동 갱신

### public 폴더의 CSV 파일 교체 시:
1. public 폴더의 기존 CSV 파일 삭제
2. 새 CSV 파일 복사
3. 브라우저 새로고침 (F5)
